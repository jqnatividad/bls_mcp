# Task Completion Checklist

When completing a task in this project:

1. **Build**: Run `npm run build` to verify TypeScript compiles without errors
2. **Test manually**: Run `npm run dev` to verify the MCP server starts correctly (no test suite exists)
3. **No linter/formatter**: There is no linter or formatter configured, so no lint/format step is needed

Note: There are no automated tests, no linter, and no formatter in this project.
